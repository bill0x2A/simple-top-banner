A simple top banner with customizable link and icons.

== Description ==
Es viernes y el cuerpo lo sabe.

Add a customizable top banner to your WordPress site with:
- Custom link text and URL
- Font Awesome icons
- Typography controls
- Color customization
- Custom CSS support

== Installation ==
1. Upload the 'top-banner' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the banner in the 'Top Banner' menu item
